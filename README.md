# Installation

* Clone the repo
  > git clone https://github.com/uchkunovdaniel/DrawML.git
* Enter the directory
  > cd DrawML
* Install the Requirements
  > pip install -r requirements.txt / python3 -m pip install requirements.txt

# Running

  * Run the app
    > uvicorn main:app
  * Open the app
    > http://127.0.0.1:8000/
